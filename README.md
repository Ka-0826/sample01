# 社員登録サイト  
## 概要  
本プロジェクトは、社員情報を管理するための 社員登録サイト です。  
社員の登録・編集・削除などの機能を提供します。  

## 動作環境  
言語: PHP  
開発環境: MAMP  
エディタ: VSCode
